What is Github?


  • GitHub is a web-based tool for collaborating on software, websites and documentation projects. It provides version control, issue tracking, and wiki space for hosted projects.
  • Github is built on top of Git, an open-source version control system that was started by Linus Trovalds.

En inglés, por favor?

  • Github is a tool for collaboratively creating and documenting projects.
  • Github allows you and your colleagues to work on the same file at the same time and then merge your versions together back into one file.
  • Github keeps a history of your project, allowing you to revert to before a particular change

Version Control 101

Github allows and your colleagues to upload your revisions of a single file to the same page, and Git will save two copies. Later, you can merge your changes together without losing any work along the way. You can even revert to an earlier version at any time, because Git keeps a "snapshot" of every change ever made.
